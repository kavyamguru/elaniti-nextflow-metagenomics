# elaniti-nextflow-metagenomics

Modular Nextflow DSL2 metagenomics pipeline with Docker and Slurm support for reproducible microbial genome analysis.

## Highlights
- DSL2 modular pipeline structure
- Container-friendly execution
- HPC-ready (Slurm)
- Reproducible workflow design

## Typical use cases
- Metagenomic preprocessing and downstream analysis
- Team-based pipeline development
- Standardized runs across local and HPC environments
